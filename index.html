What We Are About!

We are a gaming community comprised of gaming fanatics that put gaming above all us to get the full experience from each game that we play. We want to give detailed reviews from personal experience so that you can best decide on the games that you would like to play. We want to make sure that we cover any and all games from all types and genres so that we can cater to all the needs of every user joining us. We also have forms to fill out for the game of your choice in the Hot Games section of the website and would love to hear from you so we can analyze which ever game you choose. Please join us on this ride of gaming experiences so we can give a well rounded review of any and all games ahead!
